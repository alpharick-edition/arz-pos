# ARZ POS — How to Run

## First time setup (do this once)

1. Install **Node.js** from https://nodejs.org  
   → Click the big green "LTS" button and install it normally.

---

## Running the app

### On Windows:
Double-click **START.bat**  
That's it. A browser tab will open automatically.

### On Mac:
1. Open Terminal
2. Drag the arz-pos folder into the Terminal window
3. Type: `bash START.sh` and press Enter

---

## Every time after that
Just double-click START.bat again.  
It will open at: **http://localhost:5173**

---

## Stopping the app
Close the black terminal/command window, or press **CTRL+C** inside it.
